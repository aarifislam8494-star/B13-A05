Ans: Difference between var, let and const :var:   
(i) Blocked Scoped.   
(ii) same variable name can not be define again.   
(iii) old method, before ES6.
let:    
(i) block scoped (inside specific function).  
(ii) Previously define the variable value you can not reassign.
const:   
(i) You can't redeclared the value.   
(ii) you can change the re assign value from variable.
